## 🚀 What I'm Building

### Kibox — Co-Founder

> **Smart kitchen inventory & food intelligence platform powered by AI**

Kibox helps households, restaurants, and food businesses track pantry stock, reduce waste, and make smarter food decisions — all in one platform.

| Layer | What it does |
|---|---|
| 🏠 **Household Portal** | Pantry tracking, low-stock & expiry alerts, shopping lists, consumption insights |
| 🤖 **Chef AI** | GPT-4o powered culinary assistant with live pantry awareness |
| 📊 **Intelligence Dashboard** | National food consumption analytics across all 50 US states (USDA · BLS · Census data) |
| 🏢 **B2B Portal** | Restaurant & food business inventory management |
| 📱 **WhatsApp Alerts** | Real-time low-stock notifications |

**Stack:** Spring Boot · React 18 · PostgreSQL · OpenAI GPT-4o · JWT Auth · Docker · Tailwind CSS
